# 期货盈亏计算器 - 功能描述文档

## 一、产品概述

期货盈亏计算器是一款面向期货交易用户的轻量化网页工具，支持黄金、美日汇率、美原油、比特币、以太坊等主流期货品种的盈亏金额（美元）快速计算。用户可通过选择品种、设置交易手数和波动点数，实时获取精准的盈亏结果，界面简洁直观、交互流畅，兼具实用性与视觉体验。

## 二、核心功能

### 1. 多品种盈亏计算

- **支持品种**：涵盖5类主流期货品种，具体如下：

|    品种代码|    品种名称|    每标准手每点价值（美元）|    计算依据|
|---|---|---|---|
|    XAUUSD|    黄金/美元|    100|    固定点值|
|    USDJPY|    美元/日元|    6.45|    按155汇率计算（1000÷155≈6.45）|
|    USOIL|    美国原油|    10|    固定点值|
|    BTCUSD|    比特币/美元|    1|    固定点值|
|    ETHUSD|    以太坊/美元|    0.1|    固定点值|
- **计算逻辑**：盈亏金额 = 交易手数 × 波动点数 × 品种单点点值（美元/手），支持正负点数输入（正数为盈利，负数为亏损）。

### 2. 实时交互计算

- 无需点击“计算”按钮，用户选择品种、输入/修改手数/点数时，系统自动实时更新盈亏结果；

- 支持小数手数输入（最小0.01手，步长0.01），满足精细化交易计算需求；

- 结果区实时展示计算公式，清晰呈现盈亏推导过程，提升透明度。

### 3. 可视化结果展示

- 盈亏金额以大号字体突出显示，不同状态区分颜色：

    - 盈利：绿色字体；

    - 亏损：红色字体；

    - 无盈亏：灰色字体；

- 数值更新时附带轻微缩放动效，强化视觉反馈。
